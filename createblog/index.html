Hexo博客之路

发表于 2017-03-17 | 作者 干锅田鸡 |
前言

最近有人问我创建个人博客有什么方案，于是自己也就摸索了一番，最后选择了通过github page作为载体，以Hexo为实现方式，以git，npm作为工具，最后通过实践确定为切实可行的方案，于是在里记录下来

简介

Hexo:我将其理解为一个转换器，以markdown的方式记录笔记，hexo整合主题将markdown转换为html页面，最后发布到GitHub上
npm:安装一些所需依赖
git:管理本地的代码仓库，clone主题文件
github:承载博客页面代码，发布GitHub page
Next主题：个人比较喜欢的一个简洁的黑白主题，能够接入一些第三方，如评论功能
整个的流程如下：

  • npm安装hexo，初始化配置
  • git clone next主题文件
  • 创建文章
  • hexo g 将markdown生成html页面
  • hexo s 启动本地服务预览博客
  • GitHub创建仓库，并在hexo配置文件中指定仓库的ssh url
  • hexo d 将生成的页面发布到GitHub上
  • GitHub发布GitHub page
  • 配置域名，指定到GitHub page
